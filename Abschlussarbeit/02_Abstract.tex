\chapter*{Abstract}
%\addcontentsline{toc}{chapter}{Abstract}
\noindent
% TEXT ENGLISCH %
This thesis deals with the implementation of a marker-based localization system for an unmanned aerial vehicle in an indoor environment.
Satellite navigation is not available indoors, so the absolute position reference normally used for flight control is not available.
The goal of this thesis is to localize an unmanned aerial vehicle in an indoor environment without satellite navigation and to feed this position estimate 
into the flight control system as an external reference.
The process is based on \texttt{tag36h11} tags from the AprilTag family, whose positions are stored in a pre-generated map.
The map is created in a three-step, upstream process for simultaneous localization and mapping.
During flight, the onboard computer mounted on the aircraft continuously determines the aircraft’s pose based on marker detections from two cameras and transmits this information to the flight controller via the communication interface.
There, it is processed as an external position reference in the flight control system’s extended Kalman filter.
The system is built entirely from open-source software and standard components.
This thesis describes the selection of mapping and marker methods, the calibration of the cameras, the map creation process, as well as the in-flight position determination and its integration with the flight control system.
The evaluation includes three mapping experiments in which different parameterizations—each requiring varying levels of computational effort—are compared with one another and with a known reference map.
The results show that map generation is reliably possible using the parameterizations examined.
In addition, a test flight is conducted in which the aircraft flies a square trajectory with a side length of 1.5~m.
The aircraft reliably follows the specified edges.
However, continuous control movements are noticeable during flight, leading to slight deviations from the specified flight path.

\chapter*{Kurzfassung}
\addcontentsline{toc}{chapter}{Kurzfassung}
\noindent
Die vorliegende Arbeit befasst sich mit der Umsetzung eines passmarkenbasierten Lokalisierungssystems für ein unbemanntes Fluggerät im Innenraum.
In Innenräumen steht keine Satellitennavigation zur Verfügung, sodass die sonst übliche absolute Positionsreferenz für die Flugsteuerung entfällt.
Ziel der Arbeit ist es, ein unbemanntes Fluggerät in einer Innenraumumgebung ohne Satellitennavigation zu lokalisieren und diese Positionsschätzung 
als externe Referenz in die Flugsteuerung einzuspeisen.
Als Grundlage dienen Passmarken des Typs \texttt{tag36h11} aus der AprilTag-Familie, deren Lage in einer vorab erstellten Karte hinterlegt ist.
Die Karte wird in einem dreistufigen, vorgelagerten Verfahren zur simultanen Lokalisierung und Kartierung erstellt.
Im Flug bestimmt der, auf dem Fluggerät befstigte Begleitrechner, aus den Markerdetektionen zweier Kameras fortlaufend die Pose des Fluggeräts und übergibt sie über die Kommunikationsschnittstelle an den Flugregler.
Dort wird sie im erweiterten Kalman-Filter der Flugsteuerung als externe Positionsreferenz verarbeitet.
Das System ist vollständig aus quelloffener Software und gängigen Komponenten aufgebaut.
Die Arbeit beschreibt die Auswahl der Kartierungs- und Markermethodik, die Kalibrierung der Kameras, die Kartenerstellung sowie die im Flug ablaufende Positionsbestimmung und deren Anbindung an die Flugsteuerung.
Die Evaluierung umfasst drei Versuche zur Erstellung der Karte, bei denen unterschiedliche Parametrierungen mit verschieden hohem Berechnungsaufwand miteinander sowie mit einer bekannten Referenzkarte verglichen werden.
Die Ergebnisse zeigen, dass die Kartenerstellung mit den untersuchten Parametrierungen zuverlässig möglich ist.
Zusätzlich wird ein Testflug durchgeführt, bei dem das Fluggerät eine quadratische Flugbahn mit einer Kantenlänge von 1,5~m abfliegt.
Dabei folgt das Fluggerät den vorgegebenen Kanten zuverlässig.
Während des Fluges sind jedoch kontinuierliche Regelbewegungen erkennbar, die zu geringfügigen Abweichungen von der vorgegebenen Flugbahn führen.
